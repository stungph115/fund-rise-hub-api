API for website fund-rise-hub

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## à personnaliser
env.ts (stripe api secret, stripe webhook secret, APP_PASS pour utiliser gmail, NETWORK_URL pour URL de frontend en réseau local )
mailer.service.ts (gmail account name)
